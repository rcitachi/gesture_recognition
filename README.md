基于cifar10分类模型进行修改的手势识别模型，利用卷积神经网络搭建的模型对四种手势 fist、gun、Iloveyou、thumbUp 进行识别，在接收设备上显示对应的手势并控制板载LED灯进行变换，此工程基于ART-Pi创建，编译环境为RT-thread Studio。
